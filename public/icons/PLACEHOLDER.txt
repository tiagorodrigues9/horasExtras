# Ícones PWA - Placeholder

Os ícones PNG serão gerados automaticamente quando você acessar:
http://localhost:3000/gerar-icones-png.html

## Instruções:

1. Acesse http://localhost:3000/gerar-icones-png.html no navegador
2. Clique em "Download Todos os Ícones"
3. Salve os arquivos na pasta public/icons/
4. Renomeie para os nomes corretos:
   - icon-72x72.png
   - icon-96x96.png
   - icon-128x128.png
   - icon-144x144.png
   - icon-152x152.png
   - icon-192x192.png
   - icon-384x384.png
   - icon-512x512.png

## Alternativa rápida:

Para testar o PWA imediatamente, você pode:
1. Copiar qualquer imagem PNG
2. Redimensionar para os tamanhos necessários
3. Salvar com os nomes corretos na pasta public/icons/

O PWA funcionará mesmo com ícones simples!
